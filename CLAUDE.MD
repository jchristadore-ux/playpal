# Claude Project Instructions

This project must be resumable at all times.

Before making changes:
1. Read `progress.md` if it exists.
2. Read `todo.md` if it exists.
3. Determine the last completed step.
4. Continue from the first incomplete task.

While working:
- Update `progress.md` after every completed milestone.
- Save all findings and plans to files instead of keeping them only in memory.
- Never repeat completed analysis unless the source code has changed.
- Never restart the project from scratch if previous progress exists.
- Work in small, incremental steps.
- After each major change, update `todo.md` with completed and remaining tasks.

If interrupted or a usage limit is reached:
- Ensure `progress.md` contains:
  - What was completed
  - Files modified
  - Remaining tasks
  - The exact next action to perform
